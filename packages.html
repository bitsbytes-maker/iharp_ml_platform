Python Package

CMAD

v0.1.0.post6  ·  Python ≥ 3.8  ·  License: iHARP

GeoAI Anomaly Detection Climate Cryosphere Spatio-Temporal Unsupervised
pip install cmad

CMAD is a lightweight, unsupervised framework for detecting clusters of extreme negative spatio-temporal anomalies in sequential 2D gridded data. It applies convolution-based matrix operations over time-series data to surface spatially coherent anomalies — without requiring any labeled training data.

Designed for applications such as sea ice melt detection, environmental degradation, precipitation deficits, and geospatial change analysis. The framework adapts automatically to arbitrary spatial resolutions and is particularly suited for polar and high-latitude datasets where conventional methods struggle with irregular spatial structure and sparse observational records.

The package is available on PyPI and supports NetCDF and array-based inputs. It integrates seamlessly into standard Python scientific workflows using NumPy, SciPy, and xarray.
